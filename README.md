# nuxt-gitpod
A nuxt & buefy playground based on gitpod

## Usage
Just add 'gitpod.io/#' to the url: [https://gitpod.io/#github.com/silvester-pari/nuxt-gitpod/](https://gitpod.io/#github.com/silvester-pari/nuxt-gitpod/)

## Workspace Snapshot
